## Pengenalan Wajom Contact Grabber

Wajom Contact Grabber adalah extension Chrome yang membolehkan anda mengekstrak nombor telefon dari group WhatsApp dengan mudah dan selamat. Extension ini berfungsi sebagai alat bantu untuk:

- **Mengumpul Database**: Kumpul nombor telefon untuk campaign marketing
- **Pengurusan Contact**: Sedut nombor dari multiple groups sekaligus
- **Export ke CSV**: Save data dalam format CSV untuk diimport ke Wajom
- **Integration Seamless**: Terus integrate dengan sistem Wajom

## Prasyarat

Sebelum install Wajom Contact Grabber, pastikan anda ada:

- ✅ Google Chrome browser (versi terkini)
- ✅ Akaun Wajom yang aktif
- ✅ Nombor WhatsApp yang sudah join group-group
- ✅ WhatsApp Web access

## Langkah Selanjutnya

Selepas berjaya export contact ke CSV, anda boleh:

1. **Import ke Wajom List**: Upload file CSV ke dalam List Wajom (rujuk Tutorial 10: Cara Upload CSV)
2. **Buat Campaign**: Hantar mesej intro kepada contact yang baru di-grab (rujuk Tutorial 11: Cara Setting Campaign)
3. **Setup Chat AI**: Automatik reply kepada prospek baru (rujuk Tutorial 7: Cara Setting Chat AI)

Tip: Tip! Selepas save nombor telefon ke dalam file CSV anda boleh menggunakan feature "Campaign" untuk hantar mesej intro kepada prospek baru.

Tip: Gunakan nombor personal untuk grab contact dari group-group yang anda join secara genuine. Jangan grab dari group orang lain untuk menghormati privacy.

Alert: Alert! Sentiasa guna extension ini dengan bertanggungjawab dan patuhi Terms of Service WhatsApp.

## Soalan Lazim (FAQ)

**Q: Berapa banyak group boleh di-grab sekaligus?**
A: Anda boleh tick multiple groups dan export semuanya dalam satu file CSV.

**Q: Adakah selamat guna extension ini?**
A: Ya, extension ini dibuat khusus untuk Wajom users dan tidak menyimpan data peribadi anda.

**Q: Kenapa extension tidak muncul selepas install?**
A: Pastikan anda pilih folder "dist" dan bukan folder "wajom-grab". Refresh page WhatsApp Web jika perlu.

**Q: Boleh guna di browser lain selain Chrome?**
A: Buat masa ini, Wajom Contact Grabber hanya support Google Chrome.
